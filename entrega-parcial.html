Sobre el proyecto

Kivio nació como un proyecto personal: una aplicación móvil para facilitar la búsqueda de piso compartido y de compañeros de convivencia. La idea surgió de la dificultad real que existe hoy en día para encontrar no solo una habitación, sino una persona con la que realmente sea compatible vivir.

La app móvil ya existía y funcionaba antes del TFG. Para este trabajo se ha desarrollado una versión web completa del producto, adaptando toda la lógica y la base de datos existente a un entorno de navegador, con el objetivo de ampliar el alcance de la plataforma y demostrar la viabilidad técnica del proyecto en un contexto académico.

Qué hace la plataforma

Kivio permite a los usuarios registrarse con un perfil detallado de hábitos y estilo de vida. A partir de ahí pueden publicar habitaciones disponibles o anunciar que buscan compañero, buscar y filtrar anuncios de otros usuarios, ver el detalle de cada uno y contactar directamente a través de un chat integrado en tiempo real.

El sistema cruza los perfiles para mostrar primero las opciones más compatibles, teniendo en cuenta factores como horarios, hábitos de limpieza, tabaco, mascotas o presupuesto.

Tecnologías utilizadas

React 19 — interfaz de usuario
TypeScript — tipado estático
Tailwind CSS — estilos
shadcn/ui — componentes de UI
Supabase — base de datos y auth
PostgreSQL — almacenamiento
Realtime (WebSocket) — chat en vivo
Vite — bundler y build
